Open 3D Engine (O3DE) Project - Pong Implementation
Este projeto apresenta um estudo e implementação utilizando o Open 3D Engine (O3DE), um mecanismo 3D modular de código aberto e multiplataforma. O motor é licenciado pela Apache 2.0 e gerenciado pela Linux Foundation, sendo capaz de alimentar desde jogos AAA até simulações de alta fidelidade.

🚀 Sobre a Ferramenta
O O3DE é projetado para oferecer total liberdade aos desenvolvedores, sem taxas ou obrigações comerciais. Suas principais utilidades incluem:
* Renderizador Atom: Fotorrealista, multithread e extensível.
* Editor Visual: Conteúdo 3D extensível e intuitivo.Animação: Sistema de animação de personagens orientado a dados.
* Física: Motor de física em tempo real (incluindo integração com NVIDIA PhysX).
* Rede: Camada de rede de alto desempenho.

🛠️ Tecnologias e Integrações
* * Linguagens de Programação: Embora compatível com C++, C# e LUA, a linguagem predominante e utilizada no desenvolvimento deste projeto foi Python.
* * AWS (Amazon Web Services): Integração nativa com a nuvem para reduzir custos, aumentar a agilidade e inovação, permitindo acesso a recursos de computação e armazenamento sob demanda.
* * APIs (Gems): Uso de extensões conhecidas como "Gems", que permitem adicionar funcionalidades modulares ao motor.
 
  🎮 O Projeto: Pong
  O trabalho prático consistiu na criação de um nível de jogo ("Level_01") baseado no clássico Pong, utilizando as seguintes entidades e componentes:
  * Entity Outliner: Organização de elementos como Paddle 01, Paddle 02, Ball e Walls.
  * Ambiente: Uso do Atom Default Environment para renderização.
  * Interface: Implementação de MainMenu e In Game Menu.
  * Scripts: Lógica implementada via Python Scripts.
 
  📈 Exemplos de Mercado
  * O motor O3DE é a base de grandes produções, como o jogo New World da Amazon Game Studios.
